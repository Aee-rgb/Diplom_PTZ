# 🎥 Cистема автоматизации PTZ-камер

Desktop-приложение на Python для управления PTZ-камерами, автоматической калибровки, выполнения сценариев движения (скриптов) и базового трекинга объектов с сохранением всех событий в базу данных MySQL.

## ✨ Возможности

- **Управление PTZ**: Панорамирование (Pan), наклон (Tilt) и зум (Zoom) через кнопки интерфейса или клавиатуру.
- **Автоматическая калибровка**: Система автоматически определяет минимальные и максимальные значения диапазонов движения камеры и рассчитывает оптимальные шаги.
- **Движок скриптов**: Выполнение автоматических сценариев:
  - Патрулирование по горизонтали/вертикали.
  - Зум-циклы (плавное приближение/удаление).
  - Тур по заданным координатам (Preset Tour).
  - Круговое сканирование на 360°.
  - Поддержка пользовательских скриптов через JSON.
- **Интеграция с БД**: Пул соединений с MySQL 8.4 для логирования событий трекинга, выполнения скриптов и управления списком камер.
- **Трекинг объектов**: Модуль `HumanTracker` (заглушка, готовая для интеграции с YOLO/OpenCV DNN) для автоматического удержания объекта в центре кадра.
- **Современный UI**: Интерфейс на PyQt5 с адаптивной светло-синей темой оформления.

## 🛠 Технологический стек

- **Язык**: Python 3.10+
- **GUI**: PyQt5
- **Компьютерное зрение**: OpenCV (`opencv-python`), NumPy
- **База данных**: MySQL 8.4 (`mysql-connector-python`)

## 📂 Структура проекта
```text
├── app.py                  # Точка входа: инициализация БД, конфигурации и главного окна
├── config.json             # Файл конфигурации (создается автоматически)
├── core/
│   ├── camera.py           # Управление камерой через OpenCV (PTZ, калибровка)
│   └── database.py         # Менеджер пула соединений и CRUD-операции с MySQL
├── ui/
│   ├── main.py             # Главное окно приложения
│   ├── video.py            # Виджет отображения видеопотока
│   ├── calibration.py      # Виджет автоматической калибровки PTZ
│   ├── script_dialog.py    # Диалог выбора и запуска скриптов
│   ├── skript.py           # Движок выполнения скриптов (ScriptEngine)
│   ├── script.json         # Конфигурация встроенных скриптов
│   ├── db_dialog.py        # Диалог мониторинга таблиц базы данных
│   ├── setting.py          # Диалог настройки шагов управления
│   └── watch.py            # Модуль трекинга объектов (HumanTracker)
└── styles/
    └── theme.py            # Таблица стилей (QSS) для интерфейса
```

## ⚙️ Установка и настройка
1. Клонирование и зависимости
Создайте виртуальное окружение и установите необходимые библиотеки:
```text
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install PyQt5 opencv-python mysql-connector-python numpy
```
3. Конфигурация
При первом запуске файл config.json будет создан автоматически со значениями по умолчанию. При необходимости отредактируйте его вручную:
```text
{
  "database": {
    "host": "localhost",
    "user": "root",
    "password": "your_password",
    "database": "camera_tracker_db"
  },
  "controls": {
    "pan_step": 5,
    "tilt_step": 5,
    "zoom_step": 100
  },
  "camera": {
    "default_camera": 0
  }
}
```

## 🚀 Использование
Запуск приложения:
python app.py

1. Управление с клавиатуры
Когда окно приложения активно, используйте следующие клавиши для управления камерой:
Стрелки или W, A, S, D: Наклон вверх/вниз, поворот влево/вправо.
Q / E или + / -: Увеличение и уменьшение зума.
2. Калибровка камеры
Подключите камеру (кнопка "Включить").
Нажмите кнопку "Калибр.".
Нажмите "Начать калибровку". Система автоматически пройдет по крайним точкам PTZ и рассчитает диапазоны.
Нажмите "Применить результаты" или "Сохранить в конфиг".
3. Работа со скриптами
Нажмите кнопку "Скрипты".
Выберите встроенный сценарий (например, "Патруль: влево-вправо").
Настройте параметры (диапазон, задержка, количество циклов).
Нажмите "Запустить". Журнал выполнения и статус будут отображаться в реальном времени.
